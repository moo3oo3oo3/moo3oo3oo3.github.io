Guild Wars 2 Daily Crafting Tracker

  1. Log in to https://account.arena.net/applications.
  2. Click on "New Key".
  3. Check mark "inventories", "characters", and "progression".
  4. Click on "CREATE API KEY".
  5. Click on the 2 pieces of paper icon to copy the API key.
  6. Paste API key in the space below.

"Soft Refresh" reloads market price, character inventory, and bank data.

Use "Hard Refresh" when you delete or create characters. Also clears any saved API key and settings.

When you search up an item on the GW2 wiki, there is a box with links to other websites. The URL of these sites have the item ID inside them. For instance, Bowl of Candy Corn Custard's item ID is 36074.
